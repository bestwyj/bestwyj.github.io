# 诗苑：一个古诗文数据库查询应用

[简体唐诗开源库](https://github.com/chinese-poetry/chinese-poetry-zhCN)
[简体宋词开源库](https://github.com/chinese-poetry/chinese-poetry)

本项目主要是为了”诗苑”服务，快速查询定位特定位置是某个字的诗句，并支持多个条件联合查询。

[诗苑](https://xiaoce.fun/xunhualing)

目前已实现的功能：

- 同时配备大数据库和轻量数据库，既满足速查需求，也在必要的时候能提供搜索广度。

- 一键复制到剪贴板。

- 按热度排序。热门诗句速查，极速命中目标

- 数据覆盖唐诗和宋词，收录所有五言、七言片句。完整版数据超过127万条对句。

目前未实现的功能&BUG：

- 仅支持五言和七言格式，其它格律的古诗或宋词片段未收录。

- 部分字为繁体。但大多数常见字或意像均正常，大体不影响检索。

- 部分诗词与常见版本有偏差。这是由于同首诗的不同版本导致的。检索时可通过避开部分关键字来达到。
